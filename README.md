# Coin Collector Game (Unity)

A simple 3D coin collection game built with **Unity and C#**.
The player navigates through the level, collects coins, avoids enemies, and reaches the wall to win the game.

## 🎮 Gameplay

* Move the player around the level.
* Collect coins to increase your score.
* Avoid enemies or the game ends.
* Reach the wall to complete the level and win.

## ✨ Features

* Coin collection system with UI counter
* Sound effects for coins, death, and victory
* Enemy collision detection
* Game Over and Victory conditions
* Simple UI using TextMeshPro

## 🛠️ Built With

* Unity Engine
* C#
* TextMeshPro

## 📂 Project Structure

* `Player.cs` – Handles player interactions, coin collection, and collisions.
* `GameManager.cs` – Manages game states such as Game Over and Victory.
* UI – Displays coin count and game messages.

## 🚀 How to Play

1. Clone the repository.
2. Open the project in **Unity Hub**.
3. Load the main scene.
4. Press **Play** to start the game.
